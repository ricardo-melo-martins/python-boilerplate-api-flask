## Install

```bash
$ pip3 install -r ./requirements.txt
```

or with shell

```bash
$ ./bin/install
```

## Run

```bash
$ Flask --app server run --debug
```

or with shell

```bash
$ ./bin/start
```

or with debug

```bash
$ ./bin/start_dev
```

## Cache

### Application


```bash
$ ./bin/clear_cache
```
